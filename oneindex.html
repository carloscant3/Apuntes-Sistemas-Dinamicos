Transformada Inversa de Laplace - Paso a Paso

Vamos a resolver la transformada inversa de Laplace de la función:

\[ F(s) = \frac{s + 1}{s^2 (s + 2)} \]

Paso 1: Descomposición en fracciones parciales

Descomponemos \( F(s) \) en fracciones parciales:

\[ \frac{s + 1}{s^2 (s + 2)} = \frac{A}{s} + \frac{B}{s^2} + \frac{C}{s + 2} \]

Multiplicamos ambos lados por \( s^2 (s + 2) \):

\[ s + 1 = A s (s + 2) + B (s + 2) + C s^2 \]

Expandimos y agrupamos términos semejantes:

\[ s + 1 = A s^2 + 2A s + B s + 2B + C s^2 \]

\[ s + 1 = (A + C) s^2 + (2A + B) s + 2B \]

Igualamos los coeficientes de los términos semejantes:

  • Para \( s^2 \): \( A + C = 0 \)
  • Para \( s \): \( 2A + B = 1 \)
  • Para el término independiente: \( 2B = 1 \)

Resolviendo estas ecuaciones:

De \( 2B = 1 \), obtenemos \( B = \frac{1}{2} \).

Sustituyendo \( B \) en \( 2A + B = 1 \):

\[ 2A + \frac{1}{2} = 1 \]

\[ 2A = \frac{1}{2} \]

\[ A = \frac{1}{4} \]

De \( A + C = 0 \):

\[ \frac{1}{4} + C = 0 \]

\[ C = -\frac{1}{4} \]

Por lo tanto, la descomposición en fracciones parciales es:

\[ F(s) = \frac{1/4}{s} + \frac{1/2}{s^2} - \frac{1/4}{s + 2} \]

Paso 2: Aplicar la transformada inversa de Laplace

Aplicamos la transformada inversa de Laplace a cada término:

\[ \mathcal{L}^{-1}\left\{\frac{1/4}{s}\right\} = \frac{1}{4} \]

\[ \mathcal{L}^{-1}\left\{\frac{1/2}{s^2}\right\} = \frac{1}{2} t \]

\[ \mathcal{L}^{-1}\left\{-\frac{1/4}{s + 2}\right\} = -\frac{1}{4} e^{-2t} \]

Combinando estos resultados, obtenemos la transformada inversa de Laplace de \( F(s) \):

\[ f(t) = \frac{1}{4} + \frac{1}{2} t - \frac{1}{4} e^{-2t} \]

Respuesta final

La transformada inversa de Laplace de \( F(s) \) es:

\[ f(t) = \frac{1}{4} + \frac{1}{2} t - \frac{1}{4} e^{-2t} \]
